Howto - 在OmegaT中使用微软翻译

- 通过使用Microsoft Translator API，您可以在OmegaT的机器翻译窗口中使用Microsoft Translator机器翻译。<br0 />- 如需使用Microsoft Translator API，您需要在Microsoft Azure Marketplace中注册您的应用程序。OmegaT会使用您注册的“客户端ID”和“客户端密码”信息。
/>

订阅


https://datamarket.azure.com/dataset/bing/microsofttranslator

访问上述网站并“登陆”到微软 Azure Marketplace账户

*如果您还没有Azure Marketplace账户，您首先需要注册一个账户。.

-成功登入微软Azure Marketplace之后，您就能订阅使用Microsoft Translator了。.
-您可以从页面右边的区域中订阅任意内容。. 如果您想选择基础订阅（免费，每月高达两百万字符），在订阅列表的顶部点击“注册”.

注册您的应用


点击页面上方“我的账户”选项.
-点击左侧”我的账户“菜单中“DEVELOPERS”选项.

https://datamarket.azure.com/developer/applications

-在下方”已注册应用“中点击”注册“
-填写以下信息注册:
   -用户ID: 任意字符
   - 用户名:任意字符
   -用户密码：使用默认字符串
   - 重定向URI：任何URI（使用Microsoft Translator时所需的URL无效） http://dummy/
- 注册后，请检查您的应用程序是否已显示在页面的”已注册应用程序“中。.

https://datamarket.azure.com/developer/applications

OmegaT 设置


- OmegaT 将使用您注册所用的 "用户ID" 与"用户密码" .

Windows 用户:

-找到OmegaT.l4J.ini文件（位于OmegaT程序文件夹中），并使用首选的纯文本编辑器编辑下行:

-编辑OmegaT.l4J.ini之前
# Microsoft Translator credentials
#-Dmicrosoft.api.client_id=xxxxx
#-Dmicrosoft.api.client_secret=xxxxx
-编辑OmegaT.l4J.ini之后 (where Client ID: my_clientid1)
- 删除行头的”#“，并将”xxxx“替换为”用户ID“和”用户密码“"
# Microsoft Translator credentials
-Dmicrosoft.api.client_id=my_clientid1
-Dmicrosoft.api.client_secret=your Client secret
- 保存 OmegaT.l4J.ini.

Mac 用户：

-右键单击OmegaT.app并选择”显示包装内容“
-在OmegaT.app/Contents/MacOS/OmegaT.sh路径下找到文件并用你喜欢的纯文本编辑器编辑下行:

- 在编辑 OmegaT.sh之前
# Microsoft Translator credentials
#MS_CLIENT_ID="-Dmicrosoft.api.client_id=xxxxx"
#MS_CLIENT_SECRET="-Dmicrosoft.api.client_secret=xxxxx"
-在编辑 OmegaT.sh之后 (where Client ID: my_clientid1)
- 删除行头的”#“，并将”xxxx“替换为”用户ID“和”用户密码“"
# Microsoft Translator credentials
MS_CLIENT_ID="-Dmicrosoft.api.client_id=my_clientid1"
MS_CLIENT_SECRET="-Dmicrosoft.api.client_secret=your Client secret"
- 保存 OmegaT.sh.

Linux 用户

-添加以下选项以启动OmegaT (where Client ID: my_clientid1)
java -jar -Dmicrosoft.api.client_id=my_clientid1 -Dmicrosoft.api.client_secret=your Client secret OmegaT.jar

在OmegaT中使用Microsoft Translaort机器翻译


-启动OmegaT并打开你的项目.
- 选择 [选项] > [机器翻译]
- 点击 [Microsoft Translator]
-当你选择任意片段时，其翻译将显示在OmegaT的机器翻译窗口中.
   - 如果在机器翻译窗口中发现包含”错误请求“的消息，请检查”用户ID“和”用户密码“是否正确.
-通过 Ctrl+M (或者Mac上 Command+M) 将当前片段替换为机器翻译.
