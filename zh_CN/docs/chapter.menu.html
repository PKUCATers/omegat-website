1. 主菜单

OmegaT 的所有功能均可通过编辑器窗口顶部的菜单栏来调用。多数功能也可以通过键盘快捷方式调用。快捷键通过按 Ctrl 和字母键来激活。某些快捷键还包含其他按键。为了增强可读性，本手册中这些字母以大写形式出现。Ctrl 在 Windows 系统中使用， UNIX 和类 UNIX 操作系统使用 Ctrl 或 Control 键。Mac 用户应该使用 cmd+键 来代替它。"Cmd" 键就是在苹果键盘上印有 "command" 或“苹果”符号的那个键。

您可以根据自己的需要自定义现有的或增加新的快捷键。请参阅附录——自定义快捷键

该菜单本身包括下列条目：

Table 5.4. 编辑菜单

取消前一操作 Ctrl+Z 恢复前一个编辑操作发生之前的状态。如所修改的片段已经确认，则该操作无效。
重做前一操作 Ctrl+Y 恢复前一个编辑操作取消之前的状态。如所修改的片段已经确认，则该操作无效。

插入匹配或选择 Ctrl+I 将当前选中的模糊匹配插入到光标位置。如果已选中了目标片段的某个部分，则此功能会覆盖此部分。

如果某段文字被选为模糊匹配，那么被选择的部分会在光标处插入。
用机器翻译替换 Ctrl+M 使用选择的机器翻译服务提供的译文替换目标片段。如果未激活机器翻译服务（参阅下面的 菜单 > 选项），将不会有任何动作。
用源文本替换 Ctrl+Shift+R 使用源片段覆盖整个目标片段。
插入源文本 Ctrl+Shift+I 在光标位置插入源段落内容。
插入丢失源标签 Ctrl+Shift+T 在光标处插入（丢失的）源标签。
插入下一个丢失的标签 Ctrl+T 在光标处仅插入（所有丢失标签中的）一个标签。
导出选中部分 Ctrl+Shift+C 将当前选中内容导出至文本文件进行处理。如果未选中文本，将导出当前的源片段至该文件。当用户退出 OmegaT 时，为了与通常的剪贴板行为一致，该文件不会清空。导出的内容被复制到位于用户偏好设置文件的文件夹（请参阅Chapter 7, OmegaT文件和文件夹）的 selection.txt 文件中。
创建术语表条目 Ctrl+Shift+G 可让用户在缺省的术语表文件中创建条目。

如果您 (位于任何窗格中)在按Ctrl+Shift+G前，选择了文本中的一串字符 ，选中文本将根据默认粘贴到 源语术语 字段。

在项目中搜索... Ctrl+F 打开新搜索窗口。

如果您 (位于任何窗格中)在按Ctrl+F前，选择了文本中的一串字符 ，选中文本将根据默认粘贴到 查找 字段。

按 Ctrl+Shift+F显示上次打开的（而不是新打开）搜索窗。

搜索与替换... Ctrl+K 打开新的 搜索与替换窗口.

如果您 (位于任何窗格中)在按Ctrl+K前，选择了文本中的一串字符 ，选中文本将根据默认粘贴到 查找 字段。

转换大小写为 Shift+F3（查阅文本） 改变目标片段中高亮文本的大小写改变为选择项（全部小写、全部大写、首字母大写或句首字母大写）使用Shift + F3 循环选择上述四种选项。如果未选中文本， OmegaT 将选中包含了紧挨着光标右边字母的单词。
选择匹配 Ctrl+#N (#N 为 1 至 5 之间的数字) - 选择在匹配查看器中列出的第 N 个模糊匹配项，并将其替代或插入该段中。Section 2.2, “模糊匹配窗格” 描述了关于颜色代码的细节。
用作缺省译文   如果当前片段有多个可选译文，您可以标记所选择的译文为缺省译文。如果仅有一个可用译文，则该条目会显示为灰色。
创建多重译法   同样的片段在不同的语境中可能会使用不同的译文。如果当前译文不适合，则选择该菜单项并输入可选译文。
移除译文   删除译文设定片段为未翻译
设置空译文   将译文片段定义为空在目标文件中，此片段不会显示任何内容。在编辑器中，此译文片段显示为<EMPTY>
使用原文作为译文 Ctrl+Shift+S 使用此命令将以为片段记录为原文的同意译文，即使“允许译文片段与原文一致”选项（在选项/编辑操作中）并未选中。

在主窗口中可以使用以下快捷方式。当其他窗口挡在前面时，点击主窗口使它位于前方，或者按 Esc 键关闭当前窗口。

Ctrl 键在 Windows 系统中使用，UNIX 和类 UNIX 操作系统使用 Ctrl / Control 键。Mac 用户则使用 cmd+key。Cmd 键就是在苹果键盘上印有 "command" 或“苹果”符号的那个键。

用匹配替代 Ctrl+R 使用选中的匹配替换当前目标片段内容（缺省情况下选中第一个匹配）
插入匹配 Ctrl+I 向当前片段的光标位置插入选中的匹配（缺省情况下插入第一个匹配）
用原文本替换 Ctrl+Shift+R 用源文本内容替换当前的目标片段。
插入源文本 Ctrl+Shift+I 在目标片段的光标位置插入源文本内容
插入源标签 Ctrl+Shift+T 在目标片段的光标位置插入源标签
在项目中搜索... Ctrl+F 显示一个对话框以在项目中搜索
用机器翻译替换 Ctrl+M 使用原文的机器翻译的译文取代目标片段。如果未激活机器翻译服务（参阅下面的 菜单 > 选项 > 机器翻译），将不会有任何动作。
导出选中部分 Shift+Ctrl+C 将当前选中内容导出至文本文件进行处理。
创建术语表条目 Shift+Ctrl+G 可让用户在缺省的术语表文件中创建条目。
